01.00_QUE_ES_ESTE_PROYECTO
==========

HIA — Human Intelligence Amplifier es un sistema operativo de ejecución humano-IA para gobernar proyectos con foco, trazabilidad, continuidad y validación.

No es solo una app.
No es solo un chat.
No es solo una consola.

En su estado actual, HIA combina:
- CLI PowerShell como motor operativo.
- FastAPI backend como API local.
- React/Vite frontend como consola web.
- RADAR como observación del filesystem.
- BATON como continuidad operativa.
- HUMAN como documentación profunda.
- Git/GitHub como trazabilidad y rollback.
