01.00_PROPOSITO_DEL_DOCUMENTO
======================================================================

Este documento describe la estructura del sistema HIA desde el punto de vista documental.

El objetivo principal es permitir que un lector humano comprenda rápidamente
cómo están organizados los documentos dentro de HUMAN.README y cómo estos
documentos se relacionan entre sí.

Este documento no explica el sistema en profundidad.
Ese rol corresponde al documento:

00.2_HUMAN.SYSTEM.OVERVIEW.txt

Este documento tampoco describe procedimientos operativos.
Ese rol corresponde a documentos como:

02.0_HUMAN.OPERATION
06.0_HUMAN.PF0
05.0_HUMAN.CIS
HIA_HUM_0012_OPERATION.RUNBOOK

El propósito de este documento es ofrecer un mapa estructural del sistema.

El mapa permite identificar:

qué documentos existen
cómo se agrupan
qué rol cumple cada grupo
cómo se navega entre ellos
