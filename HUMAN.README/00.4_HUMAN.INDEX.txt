01.00_PROPOSITO_DEL_DOCUMENTO
======================================================================

Este documento proporciona un inventario completo de los documentos
contenidos en la carpeta HUMAN.README.

El objetivo es permitir que un lector humano identifique rápidamente:

qué documentos existen
qué rol cumple cada documento
cómo se relacionan entre sí
qué nivel de autoridad posee cada documento

Este documento no reemplaza otros documentos.

Este documento actúa como un índice de navegación del sistema.
